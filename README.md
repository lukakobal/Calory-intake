🍎 Calorie Intake App (React)

Simple React app for tracking daily meals.
Built as a beginner project to practice state, arrays, and user input handling.

🚀 Features

➕ Add meals

📋 Display list of meals

⚡ Instant UI updates with React state

🛠️ Tech Stack

React (useState)

JavaScript

CSS

📸 How it works

Type a meal (e.g. "Eggs")

Click Add

See it instantly appear in the list

📂 Project Structure

App.js → main logic (state, adding meals)

styles.css → basic styling

index.js → React root

🎯 Learning Goals

This project focuses on:

Understanding useState

Working with arrays in React

Rendering lists with .map()

Handling user input

⚡ Future Improvements

🔢 Add calories per meal

➖ Delete meals

📊 Total calorie calculation

💾 Save data (localStorage)

▶️ Run the project
npm install
npm start

This is a learning project, focused on fundamentals before building more complex apps.
